# wids25-predictive-customer-analysis

## Overview
This repository contains code for basic customer behaviour prediction, including CLV(customer lifetime value), churn value and RFM(recency, frequency and monetary) segmentation.
The src folder has python files having functions for performing the mentioned predictions
main.py uses them to calculate the values of the predictions and copile the result and store it in the data folder with the name "result_rfm_clv_churn.csv"

## Usage
The analysis can be performed on the sample dataset present in the data folder by running 
`python3 main.py` 

## Dataset 
The data set is present in the data folder by the name "online_retail_II.csv"
Link : https://archive.ics.uci.edu/ml/datasets/Online+Retail+II#
